Avengers Assemble

  1. Setup a HTML page with all the libraries required to run a Backbone.js and Handlebars.
    You can use this page if you want.
  2. Create a Model called Avenger with the following values: Name, Powers, Picture.
  3. Create an instance of the Avenger Model with your favourite Avengers character.
    Ex: Name: Hulk; Powers: Transformation, Superhuman Strength, Superhuman Leaping Ability, etc; PictureUrl: hulk.jpg
    You can pick other heroes in here.
  4. Write the HTML/CSS code for the attached image using a modular approach.
  5. Using the HTML created, create a Handlebar Template with the Model attributes.
  6. Create a view named Avanger with a render method and a reference to the Handlebar template.
  7. Create an instance of the Avenger View loading your Avenger Model and render in the page.
